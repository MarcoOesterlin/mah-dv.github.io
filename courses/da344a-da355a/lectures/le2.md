## Dagens exempel

```html
<!doctype html>
<html>
	<head>
		<title>Min första sida!</title>
		<meta charset="utf-8">
	</head>
	<body>
		<h1>Välkommen!</h1>
		<p>Till min otroligt vackra hemsida =)</p>
		<h2>Länkar</h2>
		<ol>
			<li>
				<a href="http://mah.se">Malmö Högskola</a>
			</li>
			<li>
				<a href="http://google.se">Google</a>
			</li>
		</ol>
		<h2>Bilder</h2>
		<img src="https://imgflip.com/s/meme/Cute-Cat.jpg" alt="Cute cat">
	</body>
</html>
```
